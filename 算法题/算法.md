
#### 每日算法（台阶问题）
一只青蛙一次可以跳上1级台阶，也可以跳上2级。
求该青蛙跳上一个n级的台阶总共有多少种跳法。
```python
def jumpFloor(number):
    a, b = 1, 2
    if number <= 2:
        return number
    for _ in range(number-2):
        a, b = b, a+b
    return b
```
青蛙一次可以跳一级也可以跳多级
```python
def jumpFloorII(number):
    if number <= 1:
        return number
    return 2*jumpFloorII(number-1)
```
